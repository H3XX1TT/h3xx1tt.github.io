Über mich

Ich bin ausgebildeter Fachinformatiker für Anwendungsentwicklung mit einer echten Leidenschaft fürs Programmieren. Für mich ist Softwareentwicklung nicht nur Beruf – es ist das, was mich jeden Tag antreibt.

Mein Fokus liegt auf der Backend-Entwicklung mit Java & Spring Boot sowie auf DevOps-Themen rund um Docker & CI/CD. Ich schreibe wartbaren, lesbaren Code und denke dabei immer an die nächste Person, die damit arbeiten muss.

Ich bin ständig dabei, neue Technologien zu erlernen und mich weiterzuentwickeln – weil guter Code nie fertig ist.
